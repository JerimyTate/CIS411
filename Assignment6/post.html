The mighty mighty world of Node.js

Event driven I/O Server Side Java Scripting

Node.js is a javascript server side language that improves the way the server handles requests by not blocking the I/O for requests. It can be written in any language that can compile to JavaScript

Node.js is mainly used to create web servers. PHP is a similar language but the big advantage Node.js has is that while PHP blocks while processing a request, Node.js takes those commands and operates them in parallel - This is the phenomenom known as non-blocking IO
